getDeclaredClasses()StreamType

StreamType

A Stream of Type objects representing all the declared members of this class.

Returns an Stream of Type objects reflecting all the classes and interfaces declared as members of the class represented by this ClassLike object. This includes public, protected, default (package) access, and private classes and interfaces declared by the class, but excludes inherited classes and interfaces. This method returns an array of length 0 if the class declares no classes or interfaces as members, or if this Class object represents a primitive type, an array class, or void.
